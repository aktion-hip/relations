List of shortcut keys

This page lists all of the default shortcut keys in the Relations application.

Category Command Description Key Sequence When
File Exit Exit the application Ctrl+X In Windows
Relations Activate Inspector view Set focus on Inspector view Alt+I In Windows
Relations Activate Persons view Set focus on Persons selection view Alt+P In Windows
Relations Activate Relations browser Set focus on Relations browser Alt+R In Windows
Relations Activate Terms view Set focus on Terms selection view Alt+T In Windows
Relations Activate Texts view Set focus on Texts selection view Alt+X In Windows
Relations Item delete Delete the selected item Delete Selection context in Relations
Relations Item relate Relate the selected item Ctrl+Shift+R Selection context in Relations
Relations Open Database Open embedded database Ctrl+O In Windows
Relations Bookmark Bookmark the selected item Ctrl+B Browser context in Relations
Relations Print Print the selected items Ctrl+P Browser context in Relations
Relations Show Item Shows the selected item in the browsers Enter Selection context in Relations
Relations-Inspector Save text edit Save changes Ctrl+S Inspector context in Relations
Relations-Inspector Undo text edit Undo of changes in text editor Ctrl+Z Inspector context in Relations
Relations-Browser Item center Move the selected item to the center Ctrl+C Browser context in Relations
Relations-Browser Item delete Delete the selected item Ctrl+Delete Browser context in Relations
Relations-Browser Item edit Edit the selected item Ctrl+E Browser context in Relations
Relations-Browser Item find Search items Ctrl+F In Windows
Relations-Browser Relation remove Removes the relation between this and the central item Ctrl+Shift+R Browser context in Relations
Relations-Browser Relations edit Edit the selected item's relations Ctrl+R Browser context in Relations
Relations-Browser Bookmarks Display the bookmarks view Ctrl+Alt+B In Windows
Relations-Browser Last changes Display the "Last changes" view Ctrl+Alt+L In Windows
Relations-Edit Add relations Relate the selected items Ctrl+R Associations context in Relations
Relations-Edit Remove relations Dissolve the selected associations Ctrl+D Associations context in Relations
Edit style Style bold Format text bold Ctrl+Shift+B Inspector context in Relations / Item editor context
Edit style Style italic Format text italic Ctrl+Shift+I Inspector context in Relations / Item editor context
Edit style Style underline Underline text Ctrl+Shift+U Inspector context in Relations / Item editor context
Edit style Bullet list Create bullet list Ctrl+Shift+D Inspector context in Relations / Item editor context
Edit style Numbered list Create numbered list Ctrl+Shift+N Inspector context in Relations / Item editor context
Edit style Uppercase list Create uppercase list Ctrl+Shift+O Inspector context in Relations / Item editor context
Edit style Lowercase list Create lowercase list Ctrl+Shift+L Inspector context in Relations / Item editor context
Navigate Backward History Navigate in history to previously showed item Alt+Left In Windows
Navigate Forward History Navigate in history to next showed item Alt+Right In Windows
Relations Open URL Opens selected text in the default browser's URL field Ctrl+Alt+U In Relations Main Window
