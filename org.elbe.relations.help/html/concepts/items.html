Items

Items are the building blocks of the rich context you're creating with the Relations application. Before you can create relations, you have the have items you can relate.

The Relations application provides items of three types:

Term item:

The term item is the item most easy to use. It consists of a title and additional text explaining the term. A term item can be used for concepts, ideas, tips, explanations and the like.

Text item:

A text item is used to store bibliographical information. If you have e.g. a scientific term, you can create a text item containing the information about the scientific paper this term is explained and then relate the scientific term with this text item.

You can enter bibliographical information for four types of text items: books, articles, contributions and web pages. Dependant on the type of text you choose you have to enter different information to create a complete bibliographical entry. At least you have to enter author and title. Further information you can enter is co-authors, publication year, publisher, location, journal (in case of an article) or book title (in case of a contribution), editors etc.

You can enter all your bibliographical information in the Relations application and create whole bibliographies using this tool afterwards.

Person item:

A person item is used to store biographical information. You can enter name, first name, year of birth and date and additional text. You can use a person item e.g. to create an item for a scientist and relate it with all text items containing the bibliographical information about texts this researcher authored.
