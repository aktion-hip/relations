The Last Changes View

The last changes view gives you the possibility to view your items sorted according their last created and last modified date respectively. Enter Ctrl+Alt+L or click the last changes button to display this support view.

Having created a new item, you might be interested to search and surf your Relations context to find items suitable to be related with your newly created item. You can easily do this by displaying the view displaying the last created items and drop the new item from this list on the drop target you found.

The list of last created items

If you rather like to diplay the list of last modified items, you can switch the view using the view menu.

The list of last modified items

For performance reasons, the number of items displayed in the Last Changes view is limited (default 20 items).
You can set this value on the application's preferences pages (Utilities > Preferences...).
