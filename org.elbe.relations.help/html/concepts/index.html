The Relations application: Concepts

The concept of the Relations application is to store items of three types: term items, text items and person items, and to create a rich context by relating these items.

The tools that support you creating and managing this context are:

  • Wizards to create items and databases
  • Relations browser
  • Selection views
  • Inspector view
  • Search view
  • Last changes view
  • Bookmarking items

To completely understand the application, you might want to know about the following aspects:

  • Action buttons
  • Workspace
  • Database export and backup
  • Drag and Drop
