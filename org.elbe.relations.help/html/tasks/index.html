Relations tasks

The main tasks of the Relations application are creating items and relating items (i.e. creating the item's context). The items and contexts created this way are fully editable. You can edit items, add styles to the text you've created and manage the items' contexts, i.e. add and delete relations.

In addition to these basic tasks, the application provides the following tasks:

  • Printing the items' content
  • Creating and connection a different data source
  • Reindexing the database
  • Database backup and restore
  • Export to XML - Import from XML
  • Export to Cloud
  • BibTEX export
  • Modifying shortcut keys
  • Software update
